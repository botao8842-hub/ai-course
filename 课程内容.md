# AI入门实战课（9.9元）

## 第1课：AI是什么？

### 什么是AI？
AI就是人工智能，简单说：**让机器像人一样思考和干活**。

### AI为什么是杠杆？
- 工资是**加法**：干一份活赚一份钱
- AI是**乘法**：1小时干完别人10小时的活

### 真实案例
- 别人写一篇文案2小时 → AI 5分钟
- 别人做一张图1小时 → AI 30秒
- 别人剪一个视频3小时 → AI 10分钟

### AI能做什么？
- ✍️ 写文章，写文案，写小说
- 🎨 画图，做设计，做logo
- 🎬 做视频、剪辑、特效
- 💬 回答问题，当助理
- 📊 做表格、分析数据
- 🧮 写代码、调试程序

## 第3课：AI写作实战（干货）

### 核心心法
AI写作的本质：**说清楚你要什么，给够上下文**

### 提示词模板（直接套用）

#### 短视频文案模板
```
主题：[你的主题]
目标人群：[谁会看]
风格：[幽默/专业/接地气/煽情]
字数：[200字]
要求：[突出痛点/给出解决方案/引发共鸣]
```

#### 例子
**输入：**
```
请帮我写一段短视频文案
主题：AI赚钱是普通人的机会
目标人群：想赚钱但没方向的普通人
风格：接地气、励志、有干货
字数：300字
要求：开头抓住注意力，结尾引导关注
```

**输出：**
（AI会生成完整文案）

### 进阶技巧

1. **角色扮演法**
   ```
   你是一个有10年经验的营销专家，帮我写...
   ```

2. **模仿法**
   ```
   模仿这个风格：[粘贴例子]
   帮我写一个关于[主题]的内容
   ```

3. **迭代法**
   - 第一次生成 → 提修改意见 → 第二次生成 → 再优化

### 变现案例
- 代写文案：50-200元/篇
- 批量生成：小红书笔记批量生产
- 写小说：番茄小说等平台签约

## 第4课：AI做图实战（干货）

### 可灵AI使用指南

#### 提示词公式
```
中文：[主体]在[场景][做什么]，[风格][运镜]
```

#### 常用风格词
- 动漫风格 anime
- 3D卡通 3d cartoon
- 油画风格 oil painting
- 摄影风格 photorealistic
- 搞笑 funny, comedy

#### 实战例子

| 想要的效果 | 提示词 |
|-----------|--------|
| 男人走路 | 一个男人在城市街道走路，动漫风格 |
| 两人对话 | 两个年轻人在咖啡馆聊天，3D卡通风格 |
| 美食 | 一碗热腾腾的牛肉面，摄影风格 |
| 建筑 | 未来科技感城市，赛博朋克风格 |

### 常见问题解决

1. **画面太暗** → 加 "bright, lighting"
2. **人物不像** → 加 "high detail face"
3. **动作不对** → 加具体动作 "running, fighting"

### 变现案例
- 头像/壁纸：9.9-29.9元/张
- 电商主图：50-200元/张
- 表情包制作：短视频引流

## 第5课：AI做视频（核心干货）

### 制作流程

```
写脚本 → AI生图 → AI配音 → 剪辑 → 发布
```

### 步骤1：写脚本（用AI）

提示词：
```
写一个1分钟的短视频脚本
主题：[主题]
风格：搞笑/感人/励志
要求：有反转/有共鸣
```

### 步骤2：生成画面

**推荐：可灵AI**
- 网址：klingai.com
- 每日66积分免费
- 支持：文生图、图生视频

**提示词技巧**
```
中文：一个[角色]在[场景][做动作]，[风格]
```

### 步骤3：AI配音

**剪映AI配音（推荐）**
- 免费，效果好
- 多种声音选择

**Azure TTS（高级）**
- 免费额度：100万字符
- 声音更自然

### 步骤4：剪辑

**剪映app（推荐）**
- 免费
- 功能强大
- AI功能多

### 沙雕动画制作流程

1. **写剧本**：用AI写搞笑段子
2. **生图片**：用可灵AI生成关键帧（每句台词一张）
3. **配音**：剪映AI配音
4. **剪辑**：把图片按台词顺序排列，添加字幕
5. **发布**：抖音/快手/视频号

### 我做过的视频案例
- 主题："用AI赚钱"
- 工具：可灵AI + 剪映
- 耗时：30分钟
