## 第1课：AI是什么？

### 什么是AI？
AI就是人工智能，简单说：**让机器像人一样思考和干活**。

### AI为什么是杠杆？
- 工资是**加法**：干一份活赚一份钱
- AI是**乘法**：1小时干完别人10小时的活

### AI能做什么？
- 写文章、写文案
- 画图、做设计
- 做视频、剪辑
- 回答问题、当助理

## 第3课：AI写作实战

### 提示词模板

```
请帮我写一段[短视频文案/文章/朋友圈]
主题：[你的主题]
风格：[幽默/专业/接地气]
字数：[200字]
```

### 例子
```
请帮我写一段短视频文案
主题：AI赚钱是普通人的杠杆
风格：接地气、幽默
字数：300字
```

### 关键技巧
1. **说清楚**要什么
2. **给例子**参考风格
3. **多迭代**不满意就改

## 第5课：AI做视频

### 制作流程

1. **写脚本** - 用AI写
2. **生成画面** - 用可灵AI/Pika
3. **配音** - 剪映AI配音
4. **剪辑** - 剪映app

### 沙雕动画制作

1. 用AI生成关键帧图片
2. 用剪映串联成视频
3. 添加配音和字幕
